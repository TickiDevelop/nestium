# Nestium

Мод для оптимизации и улучшения производительности Minecraft.

## О проекте

Nestium — это комплексное решение для повышения FPS и оптимизации использования ресурсов. Мод основан на **Sodium** и включает в себя различные улучшения для рендеринга и работы с памятью.

## Особенности

*   **Sodium**: Значительное повышение FPS и оптимизация рендеринга чанков.
*   **Entity Culling**: Отключение рендеринга сущностей и блочных сущностей, которые не видны игроку, для повышения производительности.
*   **Dynamic Lights**: Динамическое освещение от предметов в руках, выброшенных предметов и мобов (оптимизировано для высокой скорости работы).
*   Оптимизация использования оперативной памяти.
*   Улучшенная обработка сетевых пакетов.
*   Дополнительные настройки графики и производительности.
